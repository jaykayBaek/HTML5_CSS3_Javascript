웹을 통한 값의 전달 방식

            HTTP란? Server와 Client사이의 값들을 전송해주는 통신을 정의한 프로토콜
            HypertextText Transfer Protocol -> TCPIP 기술로 전송
            Client가 Server에 요청 (Request), Server가 Client에 응답(Response) -> 비동기화 ajax방식
            
            Client가 Server에 Request해서 Responsed 될 때 비동기식을 통해서 값들을 받아오는 방식을 AJax 방식이라고 한다.
            - AJAX : Asynchronous JavaScript And XML
            - XML : HTML과 같은 마크업언어이다. 서버와 데이터를 주고 받을 때 사용되는 대표적인 언어로서,
            동기화도된 자료를 전송하게 된다. 이때 XML은 불필요한 태그가 많이 전송되기에 가독성이 떨어진다.
            - JSON : JavaScript Object Notation으로 서버와의 전송에 필요한 데이터 포멧을 말한다.
            JSON방식은 모바일에서의 서버와 데이터 전송에도 주로 사용되고 있으며,
            프로그램의 종류나 플랫폼에 상관 없이 사용할 수 있다. (예:C,C+,C++,Java,Python,PHP,코틀린,GO언어...)
            자바스크립트에서는 객체와 같이 key와 value로 구성되어 처리된다. like Map
            처리방식은 Object를 String으로 직렬화(deserialize)해서 다시 Object로 변환해서 사용한다.
            예) 논리값을 JSON을 이용해서 string변환?
                let json = JSON.stringify(true);
                console.log(json); // 결과 'true'
                -> 참, 거짓은 parsing하지 않아도 됨.
            예) 배열을 JSON을 이용해서 string변환?
                let json = JSON.stringify(['apple', 'banana', 'monkey']);
                console.log(json); // 결과 : ["apple", "banana", "monkey"] JSON은 큰따옴표("")로 넘어온다.
