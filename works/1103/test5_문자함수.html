함수연습

        0         1         2
        0123456789012345678901234567890
        ABCDEFGHIJKLMNOPQRSTUVWXYZABCabab

length : 객체(문자열)의 길이

slice(a,b) : 지정된 문자열(a에서 b앞까지)을 추출해준다.

substring(a,b) : 지정된 문자열(a에서 b앞까지)을 추출해준다.

substr()

replace(a,b):a를 b로 치환(교체)

replaceAll(a,b):a를 b로 모두 치환

toLowerCase() : 소문자로 치환

toUpperCase() : 대문자로 치환

concat(a, b) : a와 b를 결합한다.

trim() : 양쪽 공백을 삭제한다.

trimStart() : 앞쪽 공백을 삭제한다.

trimEnd() : 뒤쪽 공백을 삭제한다.

charAt() : 자바에서 String열의 1문자를 char로 바꿔주는 것

charCodeAt() : 문자열에서 글자를 하나 꺼내서 아스키코드로 변경

문자[index] : charAt과 동일함. (js의 편리한 점)

split() : 구분자로 기준으로 잘라준다. 잘라진 각각의 요소들은 배열로 저장된다.

indexOf() : 특정문자의 위치값을 찾아준다. (찾는 값이 없으면 -1 출력)

lasIndexOf() : 특정문자의 위치를 뒤에서부터 찾아준다.

search() : 특정문자의 위치값을 찾아준다(찾는 값이 없으면 -1 출력)

match() : 찾는 문자열이 있으면 '문자열'을 반환, 없으면 공백을 반환한다.

includes() : 찾는 문자열을 포함하고 있으면 true, 아니면 false 반환
