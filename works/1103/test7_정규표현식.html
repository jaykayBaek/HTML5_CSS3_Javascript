정규식(Regular Expression)

문자열 : 

        형식 : / ~~~ /플러그인
        플러그인 :
        g(global) : 전체범위에서 처리한다.
        m(multiLine) : 여러 줄에 걸쳐서 처리한다.(textarea태그)
        s(singleLine) : 한줄만 처리(안 써도 됨)
        i : 영문인 경우 대소문자 무시
        u(unicode) : 유니코드에 대한 처리
        ex) / ~~~ / gi ? 전체 범위에서 영문에 관한 처리
        
        정규표현식을 실행하기 위해서?
            1. 사용자입력문자열.match(정규표현식변수)
            2. 정규표현식변수.test(사용자입력문자열)
